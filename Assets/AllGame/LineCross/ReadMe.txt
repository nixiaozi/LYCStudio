Read Me First :)

This a Clone of ketchapp's Circle Game available in the mobile marketplaces

The Game contains very simple logic with simple and understandable coding and all of these are very well commented and easy to uderstand on how they work

 - Note
The Template is not shipped with any third party API or SDK implemented , it's just plane game.


 - Creating new challenges

 Take out a challenge prefab out of the prefabs folder and see how it works there will be two points indicating start with green circle
 and end with red circle , now go ahead and add the new challenge sprite that you created in place of the old one , there's a polygon collider
 attached to the challenge object , reset it to create the new collider according to the new sprite. check the start and end points are alligned properly or the way you want once again.

 Now Go ahead and create a prefab of this newly created or edited challenge and add it to the generator.

 Generator is the Object named ChallengeGenerator with the script of the same name ,add the prefab to the challenges array of that script component.

 - Replacing Logo

 Open the Logo Scene

 in the canvas element there's child object named Logo , use that to replace with your logo.

 - Publishing your game

Just add the Logo scene and Game scene in the build settings and let the Logo scene at the top of the hierarchy.

Put in publisher Information , Sign It , Build It and you are ready to upload and publish

Don't forget to add an awesome Icon
